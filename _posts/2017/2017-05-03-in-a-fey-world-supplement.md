In a Fey World supplement, what playable races would you like to see? Here is a list of the races being worked on so far:<br />Awaken (animal and objects with intelligence)<br />Dwarves<br />Elves<br />Faun (satyrs included here)<br />Firbolg (human children turned fey)<br />Forlarren (when a fey and a demon/devil mate)<br />Gnomes<br />Goblin<br />Halfling<br />Magii (magically created creatures, think vivi from FF with ears)<br />Mimic<br />Minotaur<br />Pixie<br />Treekin<br />Scarecrow (already a class but will be presented as a race)<br />Shroomkin  (already a class but will be presented as a race)<br />Slimeling  (already a class but will be presented as a race)<br />
